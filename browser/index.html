forgetmenot — UK Parliament library demo

This page imports the same library used by the parl Node CLI and exposes it on window.parl. Open the JavaScript console and try things like:

await parl.members.search({ name: 'Cooper', house: 'Commons', take: 5 })
await parl.hansard.lastSittingDate({ house: 'Commons' })
await parl.commonsVotes.search({ take: 1 })
await parl.parameterisedQuery.postcodeLookup('SW1P 3JA')
await parl.sparql.query('SELECT * WHERE { ?s ?p ?o } LIMIT 1')

Note on CORS: some Parliament APIs do not send Access-Control-Allow-Origin: * and will refuse browser requests from origins other than parliament.uk. If a call here errors with a CORS message, the API is reachable but won't speak to this page directly. The same call from the Node CLI works. See getting_started.md for proxy options.
